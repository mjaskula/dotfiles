Provisioning:

- limit shell script as much as possible (do everything with ansible)
- remove all provisioning from the dot files scripts


Steps to provision new machine:
- remote execute shell script grom git that will
  - install xcode
  - install homebrew
  - homebrew install ansible
  - ansible-pull .dotfiles
    - installs other homebrew/casks
    - installs ruby gems
    - installs node shit
    - sets up OS X
    - sets up shell
    - links dotfiles
